# AI & MCP Presentation

Public presentation about 6 months with AI Coding Agents and Model Context Protocol (MCP).

## Presentations

- **prskalnik.html** - Main presentation: "6 Months with AI Coding Agents: Real Lessons from Solo Testing to Team Pilot"
- **examples.html** - Examples and live demonstrations

## View Online

This presentation is hosted on GitHub Pages: https://juregrandovec-briefd.github.io/ai-mcp-presentation-public/prskalnik.html

## Local Development

1. Clone this repository
2. Open `prskalnik.html` or `examples.html` in your browser
3. Use arrow keys to navigate slides
4. Press `S` for speaker notes
